# personal-blog

This project is a fast implementation of a personal blog.

## Features
- Easy to set up and deploy.
- Responsive design.
- Supports markdown for blog posts.

## Technologies
- JavaScript
- Node.js
- Express.js
- MongoDB

## Usage
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
4. Visit `http://localhost:3000` to view your blog.

## Installation Instructions
1. Ensure you have Node.js installed on your machine.
2. Clone the repository using `git clone <repo_url>`.
3. Navigate to the project directory using `cd personal-blog`.
4. Run `npm install` to install all required dependencies.
5. Start the server with `node server.js`.
